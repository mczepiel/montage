Tools

Ninja is an advanced HTML5 application built with Montage. It targets creative professionals who want to create HTML5 content for desktop and mobile deployment. With support for technologies such as SVG, Canvas, and WebGL you can create or import content to combine with standard HTML5 elements and Montage components all within Ninja.

Learn more: Ninja Install

Screening is a web-based test automation tool you can use to create, store, and execute test cases. You can use Screening to automate UI test cases for web-based applications using recording, scripting, and debugging functionality.
