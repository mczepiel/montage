Themes

Montage comes with the Skeleton theme. It's a basic, neutral-looking theme for HTML5 controls and inputs. It can be used as a wireframe theme for prototyping, styled with CSS to create custom Skins, or used as a starting point for creating your own themes. It also aims for broad browser support to serve as a fallback theme.
