Data Binding Example

Montage makes it easier to manage your application and UI state with data bindings. A UI component or Montage object can establish a one-way or two-way binding with another component or object. When the bound property of the bound object is updated then the source object's property is kept in sync.

HTML

<input data-montage-id="number" type="number" value="0" />
<input data-montage-id="range" type="range" value="0" />

JSON Serialization

"number": {
    "prototype": "montage/ui/input-number.reel",
    "properties": { "element": {"#": "number"} }
},
"range": {
    "prototype": "montage/ui/input-range.reel",
    "properties": { "element": {"#": "range"} },
    "bindings": { "value": {"<->": "@number.value"} }
}
